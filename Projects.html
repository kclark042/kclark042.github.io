Lunar Lander

The first Javascript game that I worked on. This is a simple game where the goal is to land a rocket safely on the planet. If the velocity is too high then you crash and lose.

Maze

This is a maze game that I worked on for my Game Development class. A new maze in generated each time and the shortest path to the goal can be displayed by pressing 'p'. You can also display where you have been by pressing 'b'. You can move the character by using the arrow keys, AWSD, or JIKL.

Break Out

This was another game that I worked on for my Game Development class. The player uses the left and right arrow keys to control the paddle. The paddle will half in size when the top layer of bricks is broken through. A second ball will appear when the player reaches 100 points.

Asteroids

This is the last game that I worked on for my Game Development class. This was my version of the classic Asteroids game. The player can map the controls but the default is 'a' for rotating left, 'd' for rotating right, and 'w' to fire, and 's' to accelerate.
